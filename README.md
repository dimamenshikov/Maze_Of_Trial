Лабиринт испытаний

1. Название проекта: «Лабиринт испытаний»
<img width="792" height="506" alt="image" src="https://github.com/user-attachments/assets/54489c6c-c534-4ef4-ac39-fa251897cd17" />

2. Жанр: Third-Person
3. Платформа: ПК
4. Управление: WASD и мышь
5. Цель уровня: Найти и вынести сокровище из лабиринта за отведённое время
6. В игре присутствует возможность сохранения прогресса, настроить качества графики, громкость музыки
<img width="792" height="506" alt="image" src="https://github.com/user-attachments/assets/17c5c4cb-3790-4eca-8ff4-3e7d731ac48a" />

Элементы геймплея

1. Перемещение персонажа:
Игрок управляет персонажем с помощью клавиш WASD для передвижения по лабиринту
Игрок управляет камерой с помощью мыши для обзора окружения
<img width="792" height="506" alt="image" src="https://github.com/user-attachments/assets/c3b1bc53-1447-487d-a04a-52ae1a448d73" />

2. Система здоровья: У персонажа есть здоровье, если оно кончится персонаж возродится в последней контрольной точке
3. Лимит времени: Время прохождения игры ограничен. Игрок должен успеть найти сокровище и выбраться из лабиринта, прежде чем истечёт отведённое время.
<img width="792" height="506" alt="image" src="https://github.com/user-attachments/assets/383d5620-004f-4fb5-8c2f-94a2a1e74d37" />

4. Головоломки и  интерактивное взаимодействие: В лабиринте есть 8 комнат в каждой головоломка, чтобы ее пройти нужно взаимодействовать с элементами окружения, открывать двери, активировать механизмы.
<img width="663" height="828" alt="image" src="https://github.com/user-attachments/assets/be18cd6f-b0c1-4be6-a834-f86f5ef456e8" />

5. Логика предметов и их сбор: На уровне есть 4 вида предметов:
Сокровище, которое нужно вынести из лабиринта
Суперботинки - с их помощью можно выйти из лабиринта
Часы - дают 5 минут времени для прохождения
Ключи - открывают двери в комнаты
<img width="792" height="506" alt="image" src="https://github.com/user-attachments/assets/99a2bd37-8000-4541-8b90-e7321c74b626" />

6. Турели: В центре лабиринта размещены вражеские турели, которые стреляют лазером по игроку, отнимая его здоровья, при обнаружении персонажа камерами. Игрок должен избегать зоны видимости камер, чтобы не быть убитым лазером
<img width="792" height="506" alt="image" src="https://github.com/user-attachments/assets/0895e00e-6e17-4777-9444-a5625761688e" />
